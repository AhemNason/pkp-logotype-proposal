Main logotype

  • Should be the first choice when representing the PKP brand (?)
  • Horizontal
  • Seen in page headers , page footers
  • Each word is separated by a line break
  • “Public Knowledge Project” takes up the full height of “PKP”

Column

  • Vertical
  • Seen in page footers
  • Each word is separated by a line break
  • Used only when we talk about the software (i.e. About page, dedicated website, presentation deck about either OJS or OMP)

Superscript

  • Horizontal
  • Seen in page headers , page footers
  • Each word is separated by a line break
  • If two lines: size up 10% 
  • If single line: size up 20% 
  • The superscript denotes “satellite” services, such as Docs, School, Publishing Services, Community Forum
  • First word of the service is raised above the ”PKP” logotype by about half of the height of the “O”

Fraction

  • Vertical
  • Seen in footers, always on bottom right
  • For in-software branding only
  • Distributed software: illustrates the software’s relationship to PKP as an organisation
